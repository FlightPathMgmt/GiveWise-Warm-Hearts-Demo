# Warm Hearts Foundation Website

This is the source code for the Warm Hearts Foundation website, a demo organization for GiveWise.

## Overview

The site highlights the foundation's mission, leadership, and giving opportunities, specifically tailored to show the integration with GiveWise for strategic giving (DAFs, Securities, Crypto).

## Structure

- **index.html**: Home page.
- **who-we-are.html**: Information about the leadership and mission.
- **donate.html**: Main donation hub.
- **strategic-giving.html**: Detailed options for strategic giving (DAFs, Stocks, Crypto).
- **get-involved.html**: Information on volunteering and other ways to engage.
- **styles.css**: Main stylesheet.
- **scripts.js**: Interactive elements (mobile menu, etc.).

## Setup

Simply open `index.html` in a web browser to view the site locally.
